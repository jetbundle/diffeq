# Contributing to The Differential Equations Odyssey

Thank you for your interest in contributing! This is an educational resource, and contributions that improve clarity, fix bugs, or add pedagogical value are welcome.

## How to Contribute

### Reporting Issues

If you find a bug, typo, or have a suggestion for improvement:

1. Check if the issue already exists
2. Create a new issue with:
   - Clear description of the problem
   - Steps to reproduce (if applicable)
   - Expected vs. actual behavior
   - Notebook and cell number (if relevant)

### Making Changes

1. **Fork the repository**
2. **Create a feature branch**: `git checkout -b feature/your-feature-name`
3. **Make your changes**:
   - Follow the existing code style
   - Add comments for complex logic
   - Ensure all widgets work correctly
   - Test your changes
4. **Commit your changes**: Use clear, descriptive commit messages
5. **Push to your fork**: `git push origin feature/your-feature-name`
6. **Create a Pull Request**

### Code Style

- Follow PEP 8 for Python code
- Use type hints where appropriate
- Add docstrings to functions and classes
- Keep widget code modular and reusable
- Ensure LaTeX rendering is correct (use double backslashes for Plotly)

### Testing

Before submitting:
- Test all interactive widgets
- Verify plots render correctly
- Check for any console errors
- Ensure citations are properly formatted

### Documentation

- Update README.md if adding new features
- Add comments explaining mathematical concepts
- Keep citations complete and accurate
